Academic Interests

High Performance Computing

I have always been curious about performance and optimality of computer systems. High performance environments have a set of the best solutions that scientist have created to extract as many flops as possible from the available hardware. I hope to contribute to this community over the next years and count myself among them.

Data Security, Protection, and Privacy

I am deeply interested in learning topics related to data security, data protection,  and data privacy. I want to research for the methodologies used from the file system and storage side to guarantee data availability, data replication and deduplication, data integrity, controlled data access, encryption, data masking, differential privacy, etc.

Distributed Storage

Storage in modern computer environments are not only greatly distributed but also increasingly heterogeneous. Each storage device has its own blueprint with characteristics such as bandwidth, latency, capacity, concurrency, etc. It is increasingly important to build system software that can keep up with hardware trends.
